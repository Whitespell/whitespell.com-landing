Whitespell.req()

Contructs a new instance of the Whitespell class

cloud

Arguments

taskString

The name of the task you'd like to trigger

[params]Object

List of parameters you'd like to send with your request

[callback]Function

Function to invoke when the request is complete.

[useCors]Boolean

When true we'll use a regular CORS request instead of sockets

Return Value

WhitespellObject

A Whitespell reference for the specified location

Example

var ws = new Whitespell('https://kvendrik.whitespell.com');

ws.req('getTodoById', {
  id: 18
}, function(res){
  console.log(res);
});
